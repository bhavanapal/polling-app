# MERN Polling App

A full-stack real-time polling application built with the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. The app allows admin to create polls, delete polls, close polls and users vote, and view real-time results with secure authentication and role-based access.

## 🚀 Features
- admin/User authentication using **JWT (JSON Web Token)**
- Role-based access control (**Admin / User**)
- Create, close, and delete polls (Admin)
- Vote on polls (User)
- Real-time vote count and result aggregation
- Allow only one vote per user per poll
- Protected routes and secure API endpoints
- Responsive UI built with React.js

  ## 🛠️ Tech Stack
